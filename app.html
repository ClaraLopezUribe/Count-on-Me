Count On Me:

The App

Loop through learning! A Javascript exercise using loops. Write a program that displays numbers from 0 and 100. Display the even numbers in bold.

Instructions:

Enter a whole number between 0 and 100 in each field. The end value must be greater than or equal to the start value for a result to display. Activate the button to run the program.
